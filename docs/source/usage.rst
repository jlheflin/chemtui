Installation
------------

To use chemtui, first configure with cmake:

.. code-block:: console

   cmake -S <path/to/source/dir> -B <desired/build/folder(such as build)>

Then compile:

.. code-block:: console

   cmake --build <desired/build/folder>

Check out the :doc:`usage` section for further information, including how
to :ref:`install <installation>` the project.
