# chemtui Documentation {#mainpage}

A chemical structure file visualizer for those who live in the terminal. This
project seeks to solve the issue having to transfer files from a non-GUI
environment to be able to view them with GUI-based tools. This project
is heavily influenced by xyzmol (can't find the project to save my
life). A similar project is [catmol](https://github.com/d-haas/catmol) by
[d-haas](https://github.com/d-haas).



## Technical Design Document

What does this sucker need to do:
  - [ ] Read in chemical structure files
    - [ ] Have a class available to utilize this information
  - [ ] Be able to visualize the structure
    - [ ] General
  - [ ] Be able to manipulate the visuals for better viewing
  - [ ] Be able to view multiple structures at the same time
  - [ ] Be able to utilize vim-motions or user-defined keybinds
