chemtui documentation
=====================

**chemtui** is a molecule viewer utility written in C++. It is made with power
users in mind, enabling molecular inspection via the command line (either
locally or through SSH)

Check out the :doc: `usage` section for further information.

.. note::

   This project is under active development.
